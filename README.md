\# Banking App



A simple front-end banking web application built with HTML, CSS, and Vanilla JavaScript.



This project simulates a basic banking system with balance tracking, transaction history, and persistent storage using the browser's LocalStorage.



---



\## 🚀 Live Demo



🔗 https://w3tigerdog.github.io/banking/



---



\## 📌 Features



\- View account owner and balance

\- Deposit funds

\- Withdraw funds (with validation)

\- Transaction history tracking

\- Change account owner

\- Reset account

\- Data persistence using LocalStorage

\- Responsive UI design



---



\## 🛠 Tech Stack



\- HTML5

\- CSS3

\- Vanilla JavaScript (ES6)

\- Browser LocalStorage API



No frameworks or external libraries were used.
